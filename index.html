Segmentação automática por comportamento histórico

Software desenvolvido do zero com inteligência embarcada que analisa o comportamento histórico de cada usuário e cria segmentações automáticas. O sistema entende padrões de uso, frequência, ticket médio e ciclo de vida — e classifica cada cliente em segmentos acionáveis sem intervenção manual.   • Dashboard de gestão — visão gerencial dos segmentos, tamanho de base e performânce por grupo

E-commerce Automação JavaScript

Motor de Recuperação
de Carrinho Abandonado

Integração WordPress + Disparo personalizado por usuário

Sistema completo de recuperação desenvolvido em JavaScript puro integrado ao WordPress. O motor rastreia o comportamento do usuário no checkout — tempo na página, itens no carrinho, valor total — e, ao detectar abandono, aciona uma sequência de disparos personalizados com conteúdo dinâmico baseado nos produtos específicos que aquele usuário deixou para trás. ão é um e-mail genérico: é uma mensagem com os produtos, preço e um CTA direto para retomar aquela compra.

  • Rastreamento de abandono — detecta intenção de saída via eventos de mouse, tempo inativo e scroll no checkout
  • Personalização real — cada mensagem é montada com os dados do usuário: nome, produto, valor exato e link direto ao carrinho
  • Sequência temporizada — fluxo de 3 toques com intervalos estratégicos (1h, 24h, 72h) para maximizar recuperação
  • Zero plugin pago — integração nativa com o banco de dados do WooCommerce via JS + PHP hooks
